# 💰 Money Guide App

副業・確定申告・NISAをAIキャラクター「ナル」が案内するWebアプリ。
**完全ローカル動作・外部API不要**。

---

## 🚀 起動方法

### 必要なもの
- **Node.js**（v18以上推奨）
  - インストールがまだなら → https://nodejs.org からダウンロード

### 手順

このフォルダ（`money-guide-app`）の中でターミナルを開き、以下を順に実行：

```bash
# 1. 依存パッケージをインストール（初回のみ・1〜2分かかります）
npm install

# 2. 開発サーバーを起動
npm run dev
```

ターミナルに表示される `http://localhost:5173` をブラウザで開けばOKです。

### VS Codeで使う場合

1. このフォルダをVS Codeで開く
2. 「ターミナル」→「新しいターミナル」を選択
3. 上記の `npm install` → `npm run dev` を実行

## ✨ 搭載機能

| タブ | 機能 |
|------|------|
| 🏠 ホーム | ヒーロー＋3つの柱＋お金が育つ4段階 |
| 💼 副業 | 8職種から3問診断＋詳細ステップ解説 |
| 📋 確定申告 | 申告チェック＋税額シミュレーター＋ガイド |
| 📈 NISA | 複利シミュレーター＋制度解説＋始め方 |
| 🤖 AI相談 | キャラクター「ナル」のFAQチャット |

### AIチャット（ナル）について
APIに接続せず、内蔵FAQで応答する仕組みです。
副業・確定申告・NISA・ふるさと納税など、約25のテーマをカバーしています。

## 🔧 ビルドして公開する

静的ファイルを生成して、Vercel・Netlify・GitHub Pagesなどで公開できます：

```bash
npm run build
# → dist/ フォルダに生成されます
```

`dist/` フォルダの中身をホスティングサービスにアップロードすれば公開完了です。

## 💡 トラブルシューティング

**`npm: command not found` と出る**
→ Node.jsがインストールされていません。https://nodejs.org からインストールしてください。

**ポート5173が使われている**
→ 別のポートで自動的に起動します。ターミナルの表示を確認してください。

**画面が真っ白**
→ ブラウザのコンソール（F12キー）を開いてエラーを確認してください。
